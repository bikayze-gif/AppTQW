## Instrucciones para copiar a Excel

1. Selecciona el contenido de cada bloque de código (entre las comillas invertidas)
2. Copia (Ctrl+C)
3. Pega en Excel (Ctrl+V)
4. Usa "Datos > Texto en columnas"
5. Selecciona "Delimitado" > Siguiente
6. Marca "Otro" y escribe el carácter: **|**
7. Finalizar
